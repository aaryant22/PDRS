
# PDRS

Plagarism Detection and Reporting System


## Installation

```bash
git clone https://github.com/roguexsubmarine/PDRS.git
cd PDRS
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=app.py
flask run
```
